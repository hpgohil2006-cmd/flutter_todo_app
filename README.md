# 📱 Premium To-Do Flutter App

## 📌 Project Overview

Premium To-Do is a Flutter mobile application developed to demonstrate **local data storage using SharedPreferences**.  
The application supports full CRUD operations (Create, Read, Update, Delete) along with local notification reminders.

This project is developed individually as part of the Academic Learning Assignment (ALA).

---

## 🎯 Objective

The objective of this project is to:

- Implement local data storage in Flutter
- Demonstrate storing, retrieving, updating, and deleting data
- Use SharedPreferences for persistent storage
- Integrate local notifications for task reminders
- Design a modern and user-friendly UI

---

## 🛠️ Technology Stack

- **Flutter**
- **Dart**
- **SharedPreferences**
- **flutter_local_notifications**
- **timezone package**

---

## ✨ Features

✔ Add new tasks  
✔ Edit existing tasks  
✔ Delete individual tasks  
✔ Clear all stored data  
✔ Mark tasks as completed  
✔ Store tasks locally  
✔ Retrieve tasks on app restart  
✔ Schedule reminders with date & time  
✔ View raw stored JSON data  
✔ Premium gradient UI design  

---

## 🔄 Application Workflow

### 1️⃣ Store Data (Create)
- User enters task title
- Data stored in List
- Converted to JSON
- Saved using SharedPreferences

### 2️⃣ Retrieve Data (Read)
- On app startup
- JSON data is decoded
- Tasks displayed in UI

### 3️⃣ Update Data
- User taps task
- Edit dialog appears
- Task updated and saved again

### 4️⃣ Delete Data
- Delete individual task
- Or clear all tasks
- Storage updates immediately

### 5️⃣ Notifications
- User selects reminder date & time
- Local notification scheduled
- Reminder appears at selected time
